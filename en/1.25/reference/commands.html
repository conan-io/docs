Commands¶

Consumer commands¶

Commands related with the installation and usage of Conan packages:

  • conan install
  • conan config
  • conan get
  • conan info
  • conan search

Creator commands¶

Commands related to the creation of Conan recipes and packages:

  • conan create
  • conan export
  • conan export-pkg
  • conan new
  • conan upload
  • conan test

Package development commands¶

Commands related to the local (user space) development of a Conan package:

  • conan source
  • conan build
  • conan package
  • conan editable
  • conan workspace
