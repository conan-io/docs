# ![emscripten_logo](images/integrations/conan-emscripten-logo.png) Emscripten

Conan provides support for cross-building for both [asm.js](http://asmjs.org) and [WASM](https://webassembly.org) (Web Assembly) targets
using [Emscripten](https://emscripten.org). This enables developers to compile C/C++ code for the browser
and other JavaScript environments.

For detailed examples and step-by-step instructions, refer to:

- [Cross-building with Emscripten](https://docs.conan.io/2//examples/cross_build/emscripten.html.md#examples-cross-build-emscripten)
